# Game "Car racing" (JavaScript)

## Интенсив от Glo Academy

В ходе интенсива создана игра "Гонки" на чистом JavaScript.
После выбора уровня сложности на поле появляется машина, которой можно управлять с помощью стрелок на клавиатуре.
По мере движения появляются другие машины, которые нужно объежать. При столкновении с ними игра останавливается.
В ходе игры ведется расчет очков. После сталкновения новый рекорд записывается в localStorage.

Так же в ходе интенсива проведено поверхносное знакомство с библиотекой Electron и показан процесс перевода файлов HTML, CSS, JS в программу для операционной системы.


От себя добавлена музыка при езде, сталкновении и картинка взрыва.
Добавленные кнопки "влево" и "вправо" для устройств с небольшой шириной экрана. Такое управление позволяет управлять машинкой на мобильных устройствах.




Команды в консоле

npm init

В package.json указать:
в "scripts": {"start": "electron ."}
в "main": "js/script.js",

Установить пакеты:
npm i -D electron@latest
npm i path url -D
npm i electron-packager -g
npm i electron-packager -D

Запускать:
npm start

electron .

electron-packager .
